What You Learned in Angular

  • Two-Way Data Binding: Used [(ngModel)] for seamless interaction between the UI and component, updating the input value in real-time.
  • Event Binding: Event binding with (click) was used for actions like adding, completing, and deleting tasks.
  • Structural Directives: Used *ngIf to show a message when no tasks are available and *ngFor to loop through and display tasks dynamically.
  • Component Communication: Defined a standalone component with "@"Component to handle logic and UI for the To-Do list.
  • Local Storage: Used local storage to persist tasks between browser sessions, ensuring tasks remain even after a page refresh.
  • CSS Styling: Applied professional, minimalistic CSS for a clean, responsive UI, with hover effects and task completion indicators.
  • CLI Commands: Used Angular CLI commands like ng new, ng generate component, and ng serve to simplify app setup and development.
  • Best Practices: Followed Angular best practices such as component organization, built-in directives, and state management to create an interactive app.
